# smart-hostel-allocation
The Smart Hostel Room Allocation System is a web-based app to manage hostel rooms efficiently. Users can add rooms with details like room number, capacity, AC, and washroom availability, view all rooms, search based on requirements, and allocate the most suitable room automatically. The app uses localStorage for data persistence and features a responsive, user-friendly UI built with HTML, CSS, and JavaScript.
